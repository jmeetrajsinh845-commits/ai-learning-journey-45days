# Explainable AI using SHAP + XGBoost


## Overview

This project explains machine learning predictions using SHAP values.


## Objective

Build a churn prediction model and explain why customers are predicted as high risk.


## Technologies

- Python
- XGBoost
- SHAP
- Machine Learning


## Features

- Model training
- Prediction
- SHAP explanation
- Feature impact analysis


## Business Value

Helps companies understand AI decisions and take better actions.
