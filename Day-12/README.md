# ML Model Deployment using FastAPI


## Project Overview

This project demonstrates how to deploy a machine learning model as an API.


## Workflow

Data
↓
Training
↓
Model Saving
↓
FastAPI API
↓
Prediction


## Technologies

Python

XGBoost

FastAPI

Scikit-Learn


## How To Run


Install dependencies:

pip install -r requirements.txt


Run API:

uvicorn src.api:app --reload



Open:

http://127.0.0.1:8000/docs


## Business Use Case

Customer churn prediction system for identifying customers who may leave.
